About the App

The Math Program is a comprehensive console-based calculator designed to assist with various mathematical computations. It features functions across multiple categories:

Financial Mathematics

  • Simple Interest
  • Compound Interest
  • Add/Remove GST
  • Simple Interest Rate Calculation
  • Percentage Decrease/Increase
  • Currency Conversion
  • Loan Payment Calculator

Algebra

  • Power/Exponentiation
  • Quadratic Equation Solver

Other Utilities

  • Speed Calculation with Time
  • Area Calculator (Circle, Rectangle, Triangle)
  • Unit Converter (Length, Weight, Temperature)
  • Prime Number Checker

This program provides accurate calculations with input validation to ensure reliable results.
